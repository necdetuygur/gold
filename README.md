# Gold

## CMD
```
cd %userprofile%\desktop
git clone https://github.com/necdetuygur/gold.git
cd gold
npm install
npm i -g %userprofile%\desktop\gold
```

## BASH
```
cd ~\Desktop
git clone https://github.com/necdetuygur/gold.git
cd gold
npm install
npm i -g ~/Desktop/gold
```

## Run
```
gold
```
